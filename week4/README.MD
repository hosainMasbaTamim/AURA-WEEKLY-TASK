## todo list cli app
a simple todo list cli app. xampp is being used as local host. run this command to run get started with the app , "/opt/lampp/bin/php todo.php". (as this system was build on a linux system the php location might be different for you, e.g. php todo.php). after running this command it will list all the executable commands to do additional operations. 
make sure xampp is hosting the todo.sql database.
